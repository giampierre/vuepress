# Guía VuePress deploying in GitHub Pages

Esta es mi guía de como publicar nuestra documentación generada con VuePress en GitHub Pages.

# ¿Qué es VuePress?

VuePress es un generador de sitios estáticos alimentado por Vue que genera páginas HTML a partir de archivos de Markdown. Esto le permite centrarse en escribir la documentación en lugar de trabajar en perfeccionar el sitio web.

En esta guía se mostrarán dos maneras de publicar nuestro proyecto en GitHub Pages.

Para esta guía, nos imaginaremos que estamos trabajando con un proyecto llamado deployvuepress

# Subir todo nuestro proyecto a GitHub

  1. Establecer base correcta proyecto/.vuepress/config.js
module.exports = {
    base: '/deployvuepress/'
}
  1. Crear una carpeta con el nombre docs en la raiz de nuestro proyecto.

Nota

El nombre de la carpeta tiene que ser docs, no otro.

  1. Compilar nuestro proyecto para producción con el siguiente comando.
$ vuepress build

Al ejecutar este comando dentro de nuestro proyecto nos generará una carpeta llamada dist dentro de la carpeta .vuepress/dist

Esta carpeta (dist) tendrá todo nuestro proyecto listo para ser desplegado a GitHub Pages

  1. copiar todo el contenido de la carpeta dist generada a la carpeta docs (la que creamos antes)
$ cp -r d:/deployvuepress/.vuepress/dist/* d:/deployvuepress/docs
  1. Eliminar la carpeta dist genereda dentro de .vuepress (opcional)

(dirigirnos dentrode la carpeta .vuepress del proyecto)

$ rm -r dist
  1. subir todo el proyecto a nuestro repositorio
    $ git init
	$ git add .
	$ git commit -m "deploy in github pages with folder docs"
	$ git push -f https://github.com/[usuario]/[repositorio].git master
  1. Activar GitHub Pages en github.com/settings, opcion folder docs

# Subir solo la carpeta compilada a GitHub

  1. Generar carpeta dist con el comando vuepress build, como se muestra a continuación.
$ vuepress build

Esta carpeta (dist) tendrá todo nuestro proyecto listo para ser desplegado a GitHub Pages

  1. navegar hasta nuestra carpeta dist generada e iniciar git para subir nuestro proyecto al repositorio.
    $ git init
	$ git add .
	$ git commit -m "deploy in github pages"
	$ git push -f https://github.com/[usuario]/[repositorio].git master:gh-pages
  1. Activar Github Pages en github.com/settings, opcion gh-pages
